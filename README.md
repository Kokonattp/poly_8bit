# Polymarket Leaderboard

Top 200 Traders on Polymarket - Clean UI inspired by predict.fun

## Deploy to Vercel

1. Push to GitHub
2. Import to Vercel
3. Deploy ✅

## Local Development

```bash
npm install
npm run dev
```

## Features

- ✅ Top 200 traders with live data
- ✅ Fallback to demo data if API fails
- ✅ Filter by period, PnL, search
- ✅ Sort by any column
- ✅ View trader positions
- ✅ Links to Polymarket
